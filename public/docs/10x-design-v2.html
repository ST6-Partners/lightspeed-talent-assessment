10x vs. Lightspeed 2x Talent Assessment

A neutral capability comparison between 10x as it exists in production today and the Lightspeed 2x Talent Assessment design (Build Plan v5). Same / 10x-gaps / 10x-only — across 10x's full talent acquisition surface. All PA-2 framing from v1 is intentionally dropped.

01Framing — two different problem sizes

The single most important thing this comparison surfaces is that 10x and the Lightspeed 2x design are not the same kind of artifact. The 2x design maps onto a thin vertical slice of 10x — apply → assess → narrow → one interview, plus comms. So the buckets are lopsided by nature, and that lopsidedness is itself the finding.

10x — enterprise platform

  • Multi-tenant production recruiting platform
  • 7 microservices, ~280 permissions
  • Owns requisition → offer lifecycle end-to-end
  • Custom-built ground-up software

Lightspeed 2x — funnel accelerator

  • Lean, single-client funnel automation
  • Greenhouse + n8n + Claude + Criteria Cognitive Aptitude Test (Criteria Cognitive Aptitude Test)
  • Automates the wide top of one funnel
  • "Integrate, don't replace" — ~50–65 hr build

03Capability map — overlap & divergence

10x and the 2x design share a core of 10 capabilities. Lightspeed 2x adds 4 things 10x doesn't have; 10x does 15+ things 2x doesn't attempt — by design.

04Synthesis

10x and the Lightspeed 2x design solve different-sized problems with opposite build philosophies. 10x is a custom, multi-tenant platform that owns the full requisition-to-offer lifecycle. The 2x design is a lean, single-client automation that accelerates the top of the funnel by wiring Claude + n8n + Criteria onto Greenhouse, reserving humans for finalists.

From 2x → 10x

  • Automated, rubric-scored early work-sample as a primary skills filter
  • The "build on tools already in use" delivery model

From 10x → 2x

  • Reference checking, compensation & offer handling
  • Structured competency scorecards + multi-tenancy
  • The natural expansion path beyond top-of-funnel automation
